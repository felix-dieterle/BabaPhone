# Privacy Policy for BabaPhone

Last updated: January 5, 2026

## Overview

BabaPhone is a baby monitor application that uses your device's microphone to monitor audio. This privacy policy explains how we handle your data.

## Data Collection

BabaPhone does NOT collect, store, or transmit any personal data to external servers by default.

### Audio Data

- Audio captured by the microphone is processed in real-time
- Audio is streamed directly between devices (parent and child)
- NO audio recordings are saved to device storage
- NO audio data is uploaded to any servers (unless using mobile data mode with backend)

### Permissions Used

- **Microphone**: Required to capture audio from the baby's room
- **Network Access**: Required to stream audio between devices
- **Location**: Required for WiFi Direct discovery (NOT used for location tracking)
- **Notifications**: Required to show monitoring status while app runs in background

## Data Transmission

### Local Network Mode (WiFi/Hotspot)
- Audio streams directly between devices on the same network
- No data leaves your local network
- No third-party servers involved

### Mobile Data Mode (if enabled)
- Audio may be relayed through a signaling server for connection setup
- No audio data is stored on the server
- All audio streaming uses peer-to-peer connections when possible

## Third-Party Services

BabaPhone does not integrate with any third-party analytics or advertising services.

## Children's Privacy

This app is designed to monitor children but does not knowingly collect any personal information from children.

## Data Security

- Audio streams are transmitted without encryption in the base version
- For production use, we recommend using this app only on trusted, private networks
- Future versions may include end-to-end encryption

## Changes to This Policy

We may update this privacy policy from time to time. We will notify you of any changes by posting the new privacy policy on this page.

## Contact

For questions about this privacy policy, please open an issue on our GitHub repository.

## Your Rights

You have the right to:
- Know what data is processed
- Stop the app's data processing at any time by closing the app
- Uninstall the app to remove all local data

## Compliance

This app processes audio data only with explicit user permission and only while actively running. Users must grant microphone permission before the app can function.
